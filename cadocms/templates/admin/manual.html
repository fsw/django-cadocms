Clear Cache

This clears the application cache.

To run faster this website is using cache layer not to query database or build styles / javascripts on each request. When you change anything in the admin panel it should be also cleared from a cache, however if this does not happen for some reason and you still see the old version on the site you can use this action to force cleaning up whole cache.

After running this command first few requests to frontend might take very long (as everything is rebuilded).

Rebuild Search Index

'Search index' is automatically updated as a background process. So if you add a word in item description it might take a while before this item can be found on frontend using this keyword.

If this wont happen after a long while or you see some strange behavior of search results you can use this option to force rebuilding of the whole index (this can take a while if you have a lot of searchable items)
